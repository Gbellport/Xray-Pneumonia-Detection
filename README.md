# Xray-Pneumonia-Detection

Datasets available at: https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia

Detection of pneumonia from chest x-ray images. Used an architecture similar to VGG models with depthwise separable convolution layers. 

**Results:** 
- Accuracy: 91.03%
- Recall: 96.67%
- Precision: 89.76%
- F1-score: 0.93


**Note:** This notebook was produced within a Kaggle kernel.
